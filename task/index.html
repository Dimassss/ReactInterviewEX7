#2 javascript

+1 sumowanie

potrzebna jest funkcja w js, która przyjmie dowolną ilość argumentów i je zsumuje

sum(1,2) => 3

sum(1,2,3) => 6

+2 duplikowanie

potrzebna jest funkcja w js, która przyjmie dowolną tablicę i zduplikuje jej wartości, zwracając nową tablicę

duplicate([1,2]) => [1,2,1,2]

duplicate([3,5,8]) => [3,5,8,3,5,8]

#3 react

klient zamówił aplikację do zarządzania krajami w których to będzie działać jego strona internentowa. wykorzystuje bazę danych, która została połączona z API pozwalającym na dodawanie/edycję/usuwanie rekordów.

API (kolekcja postman): pobierz

API zawiera metody:

  • GET [test] ([test ex])
  • POST
  • PATCH
  • DELETE

co należy zrobić?

aplikację, która pozwoli na dodawanie, modyfikowanie i usuwanie krajów wg poniższego rysunku. jeżeli masz pomysł jak to zrobić lepiej, powiedz.

Ważne:

  • w tej części skupiamy się wyłącznie na funkcjonalności, wygląd nie ma znaczenia.
  • zaprojektuj strukturę aplikacji tak aby w przyszłości była łatwo rozwijalna.
